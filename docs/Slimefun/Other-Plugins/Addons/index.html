附加

想要為你的黏液科技伺服器添加額外的趣味? 那你來到對的地方了!

🏆 附加果醬比賽​

在 2020 年時我們開始主持第一個 "黏液附加果醬比賽" 在我們的 Discord 伺服器.

我們希望讓這個活動是可再次舉行. 基本前提如下: 在果醬開始時, 我們將會宣布一個主題, 每個開發人員都在有時限的框架下製作基於該主題的黏液科技附加 - 從頭開始. 當果醬結束時, 我們將會審核它們提交的內容, 對其進行評分, 並向表現最佳的附加發放獎勵.

✒️ 製作屬於自己的附加!​

你想製作屬於你自己的黏液科技附加嗎? 查看此 開發人員指南.

如果你為黏液科技製作了一個附加並希望它在此列表列出, 只需要在我們的 Wiki 儲存庫上提出一個合併請求. 你可以在此處找到教程: 擴展Wiki.
如果無法做一個合併請求, 則只需要創建一個問題並選擇"添加"模板:

## 你已貢獻但未被列在上方

由於這個清單是由機器人生成

你可以很簡單的在 Discord 上進行請求,

並提供你的 GitHub 名稱.

或建立一個PR來給自己添加貢獻清單!

你自己的翻譯身分組 並讓自己更加的獨特!
